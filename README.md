# Microsoft Systems and Software Academy (MSSA) - Cloud Application Development (CAD)

## 📖 Overview

This repository documents my progress through the **MSSA Cloud Application Development (CAD)** learning path. It serves as both a learning archive and a portfolio of practical software development experience gained throughout the program.

The curriculum emphasizes modern software engineering practices, cloud-native development, full-stack application design, and professional development using the Microsoft technology ecosystem.
